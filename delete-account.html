Account and Data Deletion

Last updated: June 13, 2026

This page explains how users of TachoMobile 2 can request the deletion of their account-related data and synchronized cloud data.

1. Delete Data Directly from the App

Users can permanently delete synchronized cloud data directly within the App using the cloud backup deletion functionality available in the synchronization settings.

Deleting cloud backups removes the data stored in Firebase services associated with the user's synchronization account.

2. Request Data Deletion

If you would like to request deletion of data associated with Google Sign-In and Firebase synchronization, please contact the developer at:

tachomobile.support@gmail.com

Please include the email address associated with the Google account used for synchronization so that we can identify and process your request.

3. What Data Will Be Deleted

Upon request, we will delete synchronized cloud data associated with your account, including:

  • Driver activity records stored in cloud backups;
  • Synchronization metadata;
  • Backup information;
  • Device synchronization identifiers associated with your account;
  • Location information contained within synchronized activity records.

Data stored locally on your device is not automatically removed as part of this process.

4. How to Remove Local Data

You may delete locally stored data at any time by:

  • Uninstalling the application;
  • Clearing the application's storage in Android settings;
  • Deleting exported files manually;
  • Deleting cloud backups through the App.

5. Processing Time

Deletion requests are processed within a reasonable timeframe and in accordance with applicable legal obligations.

Certain technical or legal obligations may require us to retain limited information where required by law.

6. Google Sign-In and Firebase Authentication

TachoMobile 2 uses Google Sign-In and Firebase Authentication to provide optional cloud synchronization features.

Requesting deletion of synchronized cloud data does not delete the user's Google account. Only the data associated with TachoMobile 2 synchronization services will be removed.

7. Contact

If you have any questions regarding account or data deletion, please contact:

tachomobile.support@gmail.com
