Основные функции

Время вождения

Контролируйте вождение, рабочее время и перерывы в реальном времени с высокой точностью прямо из главного интерфейса приложения.

Умные уведомления

Будьте в курсе и соблюдайте сложные правила режима труда и отдыха благодаря активным и интеллектуальным системным оповещениям.

Анализ вождения

Получите доступ к подробной статистике работы, автоматическим журналам и четким операционным отчетам для удобного ведения данных.

Импорт файлов .ddd

Удобно импортируйте файлы .ddd, загруженные с карты водителя или тахографа. Приложение автоматически проанализирует архивные данные и упростит контроль времени вождения.

Управление данными

Безопасно управляйте своими журналами данных с помощью простых опций экспорта, разработанных для профессиональных и независимых перевозок.

О приложении

TachoMobile 2 — это приложение для Android, разработанное в помощь водителям для ежедневного контроля их дорожной активности.

Проект ориентирован на наглядность, простоту использования, безопасную синхронизацию в облаке и шифрование данных пользователя.

Об авторе

Приложение разрабатывается независимым программистом Ярославом Езерчаком (Jarosław Jezierczak). Проект уникальным образом объединяет многолетний практический опыт работы в транспортной сфере (включая работу водителем грузовика и оператором спецтехники) с профессиональной разработкой мобильного ПО для платформы Android.
