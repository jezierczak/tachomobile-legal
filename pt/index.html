TachoMobile 2

Assistente inteligente para motoristas profissionais. Monitorize os tempos de condução em tempo real e importe ficheiros .ddd (dados descarregados do cartão de condutor ou do tacógrafo) para analisar automaticamente o seu tempo real de trabalho e repouso.

Descarregar no Google Play
🚀 Guia de início rápido para novos utilizadores

Principais funções

Tempo de condução

Monitorize a condução, o tempo de trabalho e as pausas em tempo real com elevada precisão diretamente a partir da interface principal da aplicação.

Notificações inteligentes

Mantenha-se atualizado e cumpra os complexos regulamentos de tempo de trabalho através de alertas ativos e inteligentes do sistema.

Análise de condução

Aceda a estatísticas de trabalho detalhadas, registos automáticos e relatórios operacionais claros para manter os seus dados organizados.

Importação de ficheiros .ddd

Importe facilmente ficheiros .ddd descarregados do seu cartão de condutor ou tacógrafo. A aplicação analisa automaticamente os dados históricos para facilitar o controlo dos tempos de condução.

Gestão de dados

Gerencie de forma segura os seus registos com opções simples de exportação, concebidas para operações de transporte profissional e independente.

Sobre a aplicação

O TachoMobile 2 é uma aplicação para Android concebida para ajudar os motoristas na monitorização diária das suas atividades na estrada.

O projeto foca-se na legibilidade, simplicidade de utilização, sincronização segura na nuvem e encriptação dos dados do utilizador.

Sobre o programador

A aplicação é desenvolvida pelo programador independente Jarosław Jezierczak. O projeto combina de forma única uma vasta experiência prática de vários anos no setor dos transportes (incluindo o trabalho como motorista de pesados e operador de máquinas) com o desenvolvimento profissional de software móvel para a plataforma Android.
