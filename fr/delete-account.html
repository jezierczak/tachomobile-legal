Suppression du compte et des données

Dernière mise à jour : 13 juin 2026

Cette page explique comment les utilisateurs de TachoMobile 2 peuvent demander la suppression des données liées à leur compte et de leurs données cloud synchronisées.

1. Supprimer les données directement depuis l'application

Les utilisateurs peuvent supprimer définitivement les données cloud synchronisées directement depuis l'application en utilisant la fonctionnalité de suppression de sauvegarde cloud disponible dans les paramètres de synchronisation.

La suppression des sauvegardes cloud supprime les données stockées dans les services Firebase associées au compte de synchronisation de l'utilisateur.

2. Demander la suppression des données

Si vous souhaitez demander la suppression des données associées à la connexion Google (Google Sign-In) et à la synchronisation Firebase, veuillez contacter le développeur à l'adresse suivante :

tachomobile.support@gmail.com

Veuillez inclure l'adresse e-mail associée au compte Google utilisé pour la synchronisation afin que nous puissions identifier et traiter votre demande.

3. Quelles données seront supprimées

Sur demande, nous supprimerons définitivement les données cloud synchronisées associées à votre compte, y compris :

  • Les enregistrements d'activité du conducteur stockés dans les sauvegardes cloud ;
  • Les métadonnées de synchronisation ;
  • Les informations de sauvegarde ;
  • Les identifiants de synchronisation d'appareil associés à votre compte ;
  • Les informations de localisation contenues dans les enregistrements d'activité synchronisés.

Les données stockées localement sur votre appareil ne sont pas automatiquement supprimées dans le cadre de ce processus.

4. Comment supprimer les données locales

Vous pouvez supprimer les données stockées localement à tout moment en :

  • Désinstallant l'application ;
  • Effaçant le stockage (les données) de l'application dans les paramètres Android ;
  • Supprimant manuellement les fichiers exportés ;
  • Supprimant les sauvegardes cloud via l'Application.

5. Délai de traitement

Les demandes de suppression sont traitées dans un délai raisonnable et conformément aux obligations légales applicables.

Certaines obligations techniques ou légales peuvent nous obliger à conserver des informations limitées lorsque la loi l'exige.

6. Connexion Google et Firebase Authentication

TachoMobile 2 utilise la connexion Google (Google Sign-In) et Firebase Authentication pour fournir des fonctionnalités facultatives de synchronisation cloud.

La demande de suppression des données cloud synchronisées ne supprime pas le compte Google de l'utilisateur. Seules les données associées aux services de synchronisation de TachoMobile 2 seront supprimées.

7. Contact

Si vous avez des questions concernant la suppression du compte ou des données, veuillez contacter :

tachomobile.support@gmail.com
